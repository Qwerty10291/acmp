Ограничение по времени: 0.5 секунд
В больнице есть система, в которой ведется учет пациентов. Пациент представлен в базе в виде идентификатора и номера больничной карты. Необходимо разработать программу для поиска карт пациентов по их идентификаторам.
Входные данные:
Число N (1 < N < 100) количество пациентов. Далее идут N чисел M (1 < M < 10000000) - идентификаторы пациентов. Название файла с базой пациентов.
Выходные данные:
N чисел - больничные карты пациента.
